Statement 1 - Golang Calculator

This is a simple command-line calculator written in Go.
It takes two numbers and an operator (+, -, *, /) and prints the result.

Prerequisites

Go must be installed
Check with: go version

How to Run

Navigate to the folder: cd golang/statement-1
Run the program: go run main.go
Sample workflow

The program:

Asks for the first number
Asks for the second number
Asks for an operator (+, -, *, /)
Displays the result
Sample Run

Simple Go Calculator Enter first number: 5 Enter second number: 6 Enter the choice of operation (+, -, *, /): +

Result: 11.00
