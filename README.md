# Statement 1 - Golang Calculator

This is a simple command-line calculator written in Go.  
It takes two numbers and an operator (+, -, *, /) and prints the result.

## Prerequisites

Go must be installed  
Check with:
go version

## How to Run

1. Navigate to the folder:
cd golang/statement-1
2. Run the program:
go run main.go


## Sample workflow 
The program:
1. Asks for the first number  
2. Asks for the second number  
3. Asks for an operator (+, -, *, /)  
4. Displays the result

## Sample Run

Simple Go Calculator
Enter first number: 5
Enter second number: 6
Enter the choice of operation (+, -, *, /): +
